Welcome to the hosted space for DayDream BBS.

Current version: 2.20.0-RC

There is a code freeze in the works. There will only be slight bug fixes to the current version. The roadmap for the future involves a significant re- write to the software, changing the backend database and frontend API.

  • Installation instructions:

git clone https://github.com/ryanfantus/daydream.git

(These are admittedly pretty sparse! It's a work in progress!)

move into daydream-2.20.0 directory as root,

sh install.sh

check out docs/ directory for specifics regarding inetd, xinetd, etc

That's it! Enjoy!
